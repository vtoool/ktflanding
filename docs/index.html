FAQ

Answers for travel ops leads.

Which engines are supported?

FareSnap watches Kayak search and itinerary pages. Support for additional engines is on the roadmap—contact us to prioritise yours.

How accurate is booking-class mapping?

We prioritise your configured overrides, then fall back to cabin-based logic that mirrors Sabre defaults. The demo below uses the same formatter.

Does it handle multi-city options?

Yes. Multi-city searches with 6+ segments render correctly and continue numbering beyond 9, exactly like the live extension.

Is any data sent to servers?

No personal data leaves the browser. Analytics are optional, aggregated, and can be disabled entirely by switching the provider to none.

Can I customise status codes?

The extension defaults to SS1 but honours the status you set in the popup. The sandbox shows the default value.

How do I try Pro?

Pro will introduce shared settings, audit logs, and SLA-backed support. Click any “See Pro” button to register interest—we track anonymous demand only.

Privacy-first analytics

We ship with Plausible enabled by default and no personal identifiers. Prefer zero tracking? Flip the provider to none in analytics.js. Read the full privacy policy.
